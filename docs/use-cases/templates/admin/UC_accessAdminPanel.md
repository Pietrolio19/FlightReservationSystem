# Use Case #11 - Accesso al pannello Admin

### Brief Description
L’admin raggiunge il suo pannello di controllo per visualizzare i dati.

### Level
Admin Goal

### Actors
Admin

### Pre-Conditions
L'admin deve essere autenticato.

### Basic Flow
| Step | Descrizione                                                                                                                              |
|------|------------------------------------------------------------------------------------------------------------------------------------------|
| 1    | Il sistema carica la pagina con il pannello Admin (Mock-up #9)                                                                           |
| 2    | L'admin seleziona i dati che vuole visualizzare tramite i pulsanti: "Utenti", "Voli", "Prenotazioni", "Prenotazioni Posti", "Passeggeri" |
| 3    | Il sistema mostra all'admin i dati corrispondenti al pulsante premuto                                                                    |
| 4    | L'admin può decidere se modificare o rimuovere una entry                                                                                 |
| 5    | Il sistema aggiorna o elimina la entry selezionata                                                                                       |

### Alternative Flow
Nessuno.

### Post-Conditions
L'utente ha completato la prenotazione.
