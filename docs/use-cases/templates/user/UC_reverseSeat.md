# Use Case #7 - Scelta del posto

### Brief Description
L’utente sceglie un posto sul volo precedentemente selezionato.

### Level
User Goal

### Actors
User, Admin

### Pre-Conditions
L'utente deve essere autenticato e aver iniziato la prenotazione del volo.

### Basic Flow
| Step | Descrizione                                                                                 |
|------|---------------------------------------------------------------------------------------------|
| 1    | Il sistema carica la pagina di prenotazione posti (Mock-up #5)                              |
| 2    | L'utente selezione il posto che desidera                                                    |
| 3    | Il sistema mostra i posti selezionati                                                       |
| 4    | L'utente clicca su "Continua" per confermare la scelta                                      |
| 5    | Il sistema salva i posti scelti nella sessione                                              |
| 6    | Il sistema indirizza l'utente verso lo step di inserimento dati dei passeggeri (Mock-up #6) |

### Alternative Flow
Nessuno.

### Post-Conditions
L'utente ha selezionato i posti per la prenotazione.
