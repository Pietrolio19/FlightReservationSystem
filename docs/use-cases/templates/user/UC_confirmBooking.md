# Use Case #9 - Conferma della prenotazione

### Brief Description
L’utente visualizza i dati della prenotazione e la conferma.

### Level
User Goal

### Actors
User, Admin

### Pre-Conditions
L'utente deve essere autenticato e aver selezionato: un volo, almeno un posto e aver inserito i dati dei passeggeri.

### Basic Flow
| Step | Descrizione                                                                                    |
|------|------------------------------------------------------------------------------------------------|
| 1    | Il sistema carica la pagina di conferma (Mock-up #7)                                           |
| 2    | L'utente conferma i dati della prenotazione cliccando sul pulsante "Conferma"                  |
| 3    | Il sistema salva tutti i dati relativi alla prenotazione                                       |
| 4    | Il sistema attende 3 secondi e poi indirizza l'utente alla pagina di ricerca voli (Mock-up #1) |

### Alternative Flow
Nessuno.

### Post-Conditions
L'utente ha completato la prenotazione.
