## Use Case #3 – Logout

### Brief Description
L’utente termina la sessione attiva e viene disconnesso dal sistema.

### Level
User Goal

### Actors
User, Admin

### Pre-Conditions
L’utente deve essere autenticato.

### Basic Flow
| Step | Descrizione                                               |
|------|-----------------------------------------------------------|
| 1    | L’utente seleziona l'opzione di logout                    |
| 2    | Il sistema termina la sessione corrente                   |
| 3    | Il sistema mostra la finestra di ricerca voli (Mockup #1) |

### Alternative Flow
Nessuno


### Post-Conditions
La sessione dell'utente viene terminata e il sistema non riconosce più l'utente come autenticato.
