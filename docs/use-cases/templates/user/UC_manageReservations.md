# Use Case #10 - Eliminazione di una prenotazione

### Brief Description
L’utente elimina una prenotazione.

### Level
User Goal

### Actors
User, Admin

### Pre-Conditions
L'utente deve essere autenticato e avere effettuato almeno una prenotazione.

### Basic Flow
| Step | Descrizione                                                                                      |
|------|--------------------------------------------------------------------------------------------------|
| 1    | Il sistema carica la pagina di Gestione delle Prenotazioni (Mock-up #8)                          |
| 2    | L'utente può filtrare le prenotazioni tramite i pulsanti "Tutte", "Attive", "Cancellate"         |
| 3    | L'utente sceglie una prenotazione da cancellare e la elimina cliccando sul pulsante "Cancella"   |
| 4    | Il sistema elimina la prenotazione e ricarica la pagina (Mock-up #8) per mostrare l'eliminazione |

### Alternative Flow
Nessuno.

### Post-Conditions
L'utente ha eliminato una prenotazione.
