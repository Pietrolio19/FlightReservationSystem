# Use Case #4 - Visualizzazione Profilo

### Brief Description
L'utente visualizza le informazioni legate al suo profilo.

### Level
User Goal

### Actors
User, Admin

### Pre-Conditions
L'utente deve aver eseguito il log in.

### Basic Flow
| Step | Descrizione                                                                                                        |
|------|--------------------------------------------------------------------------------------------------------------------|
| 1    | L’ospite clicca sul pulsante "Profilo"                                                                             |
| 2    | Il sistema mostra la finestra del profilo (Mock-up #4)                                                             |
| 3    | Nella finestra del profilo sono mostrate le prenotazioni dell'utente, i suoi dati anagrafici e i suoi companions   | 

### Alternative Flow
Nessuno.

### Post-Conditions
L'utente visualizza le informazione del suo profilo.
